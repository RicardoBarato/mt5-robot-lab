# Code of Conduct

MT5 Robot Lab is a technical research project. Community behavior should stay
professional, factual and safe for public collaboration.

## Expected Behavior

- Respect other users and contributors.
- Discuss risks and limitations clearly.
- Share reproducible information when possible.
- Keep public artifacts free of private data.
- Report misleading or unsafe material.

## Unacceptable Behavior

- Harassment or personal attacks.
- Posting credentials, private account data or private broker logs.
- Publishing misleading results.
- Presenting backtests as live trading approval.
- Promising profit.
- Selling signals using the official MT5 Robot Lab name.
- Implying official validation when no official validation exists.

## Enforcement

Maintainers may remove issues, comments, submissions or pull requests that
violate this code of conduct or public safety rules.
