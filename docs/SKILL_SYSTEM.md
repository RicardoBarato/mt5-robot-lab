# Skill System

The local skill system gives Codex repeatable instructions for Grand MVP work.

## Required Skills

- `mvp-grand-planner`
- `mvp-decomposer`
- `codex-scope-guard`
- `validator-release-guard`
- `github-pr-manager`
- `windows-ui-polisher`
- `mt5-adapter-architect`
- `symbol-discovery-engineer`
- `champion-dna-builder`
- `export-report-builder`
- `packaging-installer-planner`

## Skill Contract

Each skill has:

- name;
- purpose;
- when_to_use;
- inputs;
- outputs;
- hard_rules;
- validation;
- handoff.

All skills repeat the same forbidden-project rules so that large MVPs do not
leak into `ea-xau`, PayoffGrid, ONPN11 or private files.
