# Brand and Trademark Policy

This document defines the project brand boundary for MT5 Robot Lab. It is a
project policy proposal, not final legal advice.

## MT5 Robot Lab

`MT5 Robot Lab` is the product and project name.

The project maintainer controls the official public identity of:

- MT5 Robot Lab;
- MT5 Robot Lab Rankings;
- Official Verified;
- Champion DNA Verified;
- future official badges, categories and trust labels.

## Forks and Derivatives

Forks may use the source code according to the applicable license, but forks
should not present themselves as the official MT5 Robot Lab ranking.

If a fork changes the public distribution in a meaningful way, it should use a
different product name and clearly state that it is not the official MT5 Robot
Lab project.

## Official Ranking Names

The following names should be treated as official trust signals controlled by
the project:

- `MT5 Robot Lab Rankings`;
- `Official Verified`;
- `Champion DNA Verified`;
- similar badge names that imply official validation.

These names should not be used to mislead users into thinking a fork, broker,
signal group or paid product is endorsed by the official project.

## Prohibited Brand Uses

Do not use the brand to:

- imply live trading approval;
- imply broker affiliation;
- imply MetaQuotes affiliation;
- imply an investment recommendation;
- sell signals using the official project identity;
- hide modified code behind official labels;
- mislead users about validation status.

## Acceptable References

Acceptable wording:

```text
Built with MT5 Robot Lab
Submitted to MT5 Robot Lab leaderboard
Research backtest artifact
```

Avoid wording:

```text
official profitable robot
approved live strategy
official signal service
```
